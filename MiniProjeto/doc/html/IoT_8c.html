Detailed Description

Implementation file for the "smart" I/O module.

This file contains the main implementation for an IIoT device using Zephyr RTOS. It handles UART communication, button states, LED control, and analog sensor readings.

void thread_analog_sensor ( void  )

Thread for reading and processing the analog sensor value.

This function reads the raw value from the analog sensor using the ADC, converts this value to millivolts, and stores both values in the real-time database (RTDB).
