Detailed Description

Header file for a Doubly Linked List (DLL) module.

This module provides declarations for functions to manipulate a Doubly Linked List (DLL). It includes functions for initialization, insertion, removal, searching, displaying elements, random filling, and clearing the DLL.

Randomly fills the DLL with nodes.

This function randomly generates keys and data arrays for nodes and inserts them into the DLL until it is full. The keys are generated randomly, and the data arrays contain random alphanumeric characters. If the DLL is already full, no new nodes are inserted.
