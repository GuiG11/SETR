Detailed Description

Main function for the vending machine project.

This main file initializes the vending machine and configures the buttons and LEDs. It defines the callback functions for button presses and handles the corresponding actions. Additionally, it provides a function to print the current status of the vending machine.

The vending machine can be controlled with four buttons and there are four status LEDs. Button 1 is used to insert 1 credit, button 2 is used to insert 2 credits, button 3 is used to browse products, and button 4 is used to confirm a product selection. LEDs indicate the status of the vending machine, such as product selected or product/refund availability.
